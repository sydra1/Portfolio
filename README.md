# Sidra Zulfiqar — Personal Portfolio

A responsive personal portfolio website built from scratch using pure HTML, CSS, and JavaScript. No frameworks, no libraries.

## Live Demo
🔗 (https://sidrazulfiqar1.netlify.app/)

## Sections
- **Hero** — Headline and call-to-action
- **About** — Background and key project metrics
- **Skills** — Categorized tech stack
- **Projects** — 6 real projects with results and tech stacks
- **Contact** — Email, LinkedIn, GitHub, phone

## Features
- Scroll-reveal animations using Intersection Observer API
- Active navigation highlighting on scroll
- Fully responsive on mobile and desktop
- Zero dependencies — pure HTML/CSS/JS

## Tech Stack
HTML · CSS · JavaScript
